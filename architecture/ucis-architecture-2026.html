// Key Concepts

What Makes It Think

🧬
Session DNA
Automatic classification of work patterns — explorer, builder, debugger, or mixed. The system knows how you're thinking before you tell it.
📈
Topic Momentum
What's rising in attention vs fading. Tracked across sessions, not within them. The system detects shifts before you're conscious of them.
⚡
Memory Velocity
Rate of memory creation, update, and consolidation. A signal of learning rate — high velocity means breakthrough territory.
🌐
Cross-Domain Traversal
Graph queries that find paths between a personal breakthrough, a technical pattern, and a research discovery. Insight lives at intersections.
🌊
Ambient Intelligence
The system is always processing — scoring new docs against active work, evaluating research against experience. Context you didn't ask for.
🔄
Evolutionary Scar Tissue
Every validation gate exists because something went wrong. The system learns from failures and accumulates institutional wisdom about its own blind spots.
